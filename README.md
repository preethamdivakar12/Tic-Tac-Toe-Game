# Tic-Tac-Toe Game

This is a classic Tic-Tac-Toe game implemented in Python, providing a fun two-player gaming experience.

## How to Play

1. Clone this repository to your local machine.
2. Run the `tic_tac_toe.py` file in your Python environment.
3. Players take turns marking 'X' and 'O' on a 3x3 grid.
4. The game ends when one player forms a horizontal, vertical, or diagonal line of their symbol or when the board is full.
5. To make a move, enter the row and column (e.g., "1,2" for the top center position).
6. The game will announce the winner or declare a draw.

## Features

- Simple text-based interface.
- User input validation to prevent invalid moves.
- Win-detection logic.
- Easy-to-understand Python code, suitable for beginners.

## Requirements

- Python 3.x
